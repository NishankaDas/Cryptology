# 🔐 Cryptology Programming 

This repository contains assignment programs developed for the course:

**Cryptology (PGCMSC CC 2.5)**  
Master’s Program in Computer Science  
Ramakrishna Mission Vivekananda Centenary College, Rahara  

---

## 📌 Overview

This repository focuses on the implementation of **classical and modern cryptographic algorithms** using the C / Python programming language. It covers foundational concepts in **encryption, number theory, and secure communication**, along with practical implementations.

---

## 🧠 Learning Outcomes

- Understanding classical cryptographic techniques (Shift Cipher)
- Applying modular arithmetic in cryptography
- Implementing secure communication using sockets
- Learning core number-theoretic algorithms
- Data Encryption Standard
- Advanced Encryption Standard
- RC4
- RSA
